# Ferme du Rebond - Bilan Hydrique

Application PWA de gestion de l'irrigation pour La Ferme du Rebond.

## Données
- ETP prevision : Open-Meteo
- Precipitations mesurees : Sencrop station 122872

## Installation Android
Ouvrir dans Chrome sur Android > Menu (3 points) > Ajouter a l ecran d accueil
